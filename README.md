## 📌 Sobre o projeto

O **Site CEMIL** é uma aplicação web desenvolvida com o objetivo de fornecer uma interface clara e acessível para usuários, contendo informações institucionais, serviços e comunicação com o público.

Este projeto foi desenvolvido visando:

- Organização de conteúdo institucional
- Facilidade de navegação
- Design responsivo
- Performance e simplicidade
